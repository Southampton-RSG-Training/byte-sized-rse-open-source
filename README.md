# Byte-Sized RSE: Working with Open Source Software

This is a Byte-Sized RSE session that explores how to work with open source software:

- how to assess the suitability of open source software
- how to open source your own software projects
- how to contribute to third-party open source projects

It includes a work-along example of how to make a pull request against an open source project.

## Instructor Notes

In order to run this course, you will need to make your own copy of the template code repository to act as the "third party" open source project that the attendees will make pull requests against.

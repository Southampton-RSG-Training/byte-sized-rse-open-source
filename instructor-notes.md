In order to run this course, you will need to make your own copy of the template code repository to act as the "third party" open source project that the attendees will make pull requests against.

Before the start of class go to https://github.com/Southampton-RSG-Training/byte-sized-rse-open-source-example and click on "Use this template" -> "Create a new repository".

Make sure you use your copy of the repository in class, not the original.
